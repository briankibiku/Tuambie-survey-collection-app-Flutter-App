# Tuambie

## What is Tuambie App
Tuambie is a swahili name meaning "Let us know". The app is intened to be used for creating surveys and having users fill them up.

## What features does the app have?

### Admin SignIn/SignUp
### Allow admin to create surveys
### Allow admin to view survey reports
### Allow of user to take part in several surveys
